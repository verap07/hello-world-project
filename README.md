# Hello World App

A modern iOS application built with **SwiftUI** and **SwiftData** to demonstrate seamless data persistence and a native user experience.

## 🚀 3 Must-Have Features

1.  **Persistent Data Storage**
    Leverages Apple's latest **SwiftData** framework to ensure that every item added is saved locally. Even after a hard app restart, your timestamps remain safe and accessible.

2.  **Adaptive Navigation Layout**
    Uses `NavigationSplitView` to provide a professional, multi-column interface that adapts automatically between iPhone (stacking) and iPad/Mac (sidebar and detail view).

3.  **Interactive List Management**
    Includes built-in support for standard iOS gestures, allowing users to:
    * **Add** entries instantly with the `+` toolbar button.
    * **Delete** entries using the classic "swipe-to-delete" gesture.
    * **Bulk Edit** using the native `EditButton` for rapid list organization.
